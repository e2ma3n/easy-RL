Method Descriptions
Get POST and GET are two HTTP request methods. GET is usually intended to retrieve some data, and is expected to be idempotent (repeating the query does not have any side-effects) and can only send limited amounts of parameter data to the server.
Post In computing, POST is one of many request methods supported by the HTTP protocol used by the World Wide Web. By design, the POST request method requests that a web server accept and store the data enclosed in the body of the request message. It is often used when uploading a file or submitting a completed web form.
